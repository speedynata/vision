# 🐟 V.I.S.I.O.N. 
**Visual Intelligence System for IoT & Optimized Nutrition**

---

## 📌 Overview
V.I.S.I.O.N. adalah sistem terintegrasi berbasis AIoT (*Artificial Intelligence of Things*) yang dirancang khusus sebagai langkah preventif untuk pemantauan dan manajemen budidaya kolam ikan lele. 

Sistem ini diimplementasikan untuk operasional di Awdy Farm, menggabungkan analitik kecerdasan visual dengan sensor IoT untuk mengumpulkan data empiris, memastikan kualitas lingkungan kolam, dan mengoptimalkan efisiensi nutrisi melalui auto feeding pakan secara *real-time*.

## 🚀 Key Features 
* **Auto Feeding:** Mekanisme pemberian pakan secara otomatis.
* **Real-time IoT Monitoring:** Pemantauan metrik kualitas air kolam (pH dan sisa pakan) secara terus-menerus.
* **Visual Intelligence:** Integrasi model AI untuk deteksi preventif dan analisis kondisi riak air sebagai klasifikasi kebutuhan pakan ikan.
* **Optimized Nutrition:** Manajemen pemberian pakan yang dikalkulasi berdasarkan data logis untuk meminimalisir pembuangan dan memaksimalkan pertumbuhan.
* **Mobile Dashboard:** Antarmuka responsif dan intuitif yang dibangun untuk akses pemantauan jarak jauh kapan saja.

## 🛠️ Tech Stack
* **Frontend: Flutter (Dart)
* **Backend & Database: Flask (python) & Supabase - PostgreSQL
* **Hardware / IoT: ESP32, ESP32 CAM, Sensor Ultrasonik, Sensor pH, Servo
* **AI / Data Processing: Image Classification -> Convolutional Neural Network -> MobileNetV2
