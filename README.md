# 🐟 V.I.S.I.O.N. 
**Visual Intelligence System for IoT & Optimized Nutrition**

---

## 📌 Overview
V.I.S.I.O.N. adalah sistem terintegrasi berbasis AIoT (*Artificial Intelligence of Things*) yang dirancang khusus sebagai langkah preventif untuk pemantauan dan manajemen ekosistem kolam ikan lele. 

Sistem ini diimplementasikan untuk operasional di Awdy Farm, menggabungkan analitik kecerdasan visual dengan sensor IoT untuk mengumpulkan data empiris, memastikan kualitas lingkungan kolam, dan mengoptimalkan efisiensi nutrisi pakan secara *real-time*.

## 🚀 Key Features
* **Real-time IoT Monitoring:** Pemantauan metrik kualitas air kolam (pH, suhu, dll) secara terpusat dan terus-menerus.
* **Visual Intelligence:** Integrasi model AI untuk deteksi preventif dan analisis kondisi aktual kolam.
* **Optimized Nutrition:** Manajemen pemberian pakan yang dikalkulasi berdasarkan data logis untuk meminimalisir pembuangan dan memaksimalkan pertumbuhan.
* **Mobile Dashboard:** Antarmuka responsif dan intuitif yang dibangun untuk akses pemantauan jarak jauh kapan saja.
